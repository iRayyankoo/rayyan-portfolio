# 📸 مجلد المعارض - Gallery Folder

ضع صور المعارض داخل هذا المجلد مباشرة.

## طريقة الإضافة

1. ضع الصورة هنا باسم واضح (بالإنجليزية بدون مسافات)
   مثال: `exhibition-ts-2024.jpg` أو `event-branding-01.jpg`
2. افتح ملف `index.html` وابحث عن قسم `<!-- Gallery Section -->`
3. انسخ بلوك الصورة وعدّل المسار والوصف

## أبعاد الصور الموصى بها
- الحجم: أي أبعاد — الكود يتعامل معها تلقائياً
- الجودة: JPG بجودة 80-90% للحصول على أفضل توازن بين الجودة والسرعة
- الحد الأدنى: عرض 800px

## هيكل المجلد
```
assets/
└── gallery/
    ├── README.md         ← هذا الملف
    ├── event-01.jpg      ← صور المعارض
    ├── event-02.jpg
    └── ...
```
