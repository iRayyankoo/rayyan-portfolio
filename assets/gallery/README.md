# 📸 مجلد المعارض - Gallery Folder

ضع صور وفيديوهات المعارض داخل هذا المجلد مباشرة.

## طريقة إضافة صورة
1. ضع الصورة هنا باسم واضح (مثل: `event-01.jpg`).
2. في ملف `index.html` ابحث عن `<!-- Gallery Grid -->`.
3. انسخ كود أي صورة موجودة وعدّل المسار والوصف.

## طريقة إضافة فيديو
1. ضع ملف الفيديو (MP4) هنا، مثال: `event-video.mp4`.
2. ضع صورة غلاف (Thumbnail) للفيديو، مثال: `event-video-thumb.jpg`.
3. في ملف `index.html` ابحث عن `<!-- Gallery Grid -->`.
4. انسخ كود الفيديو (الموجود في المثال رقم 6) وعدّل المسارات `data-video` و `src`.

## أبعاد الصور الموصى بها
- الحجم: أي أبعاد — الكود يتعامل معها تلقائياً.
- الجودة: JPG بجودة 80-90% لسرعة التحميل.
